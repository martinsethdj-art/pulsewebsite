# Pulse Website Pricing Page

This project is a responsive pricing page inspired by the Pulse app website. It was built as part of my front-end development training to practice layout structure, styling, and responsiveness using HTML and CSS.

## 🔧 Technologies Used

* HTML5
* CSS3 (Flexbox, media queries)
* VS Code
* Git & GitHub

## 🎯 What I Focused On

* Building a clean and structured layout
* Creating responsive pricing cards for different screen sizes
* Practicing spacing, alignment, and visual hierarchy
* Implementing hover effects and basic UI interactions

## 📱 Responsive Design

The layout adapts across:

* Desktop
* Tablet
* Mobile devices

## 📁 Project Structure

* `index.html` – Main page structure
* `styles.css` – Styling and layout
* `images/` – Logos and assets

## 🧠 Key Takeaways
- Built a responsive pricing layout from a real-world design reference
- Strengthened understanding of Flexbox and layout structure
- Improved attention to spacing, alignment, and visual hierarchy
- Practiced full workflow: development → Git → deployment
